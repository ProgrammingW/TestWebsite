Roadmap de Desarrollos



Proyecto 1: Videojuego 2D Pixelart

Un emocionante videojuego de plataformas en 2D con estilo pixelart. El protagonista es un caballero que se embarca en una épica aventura para rescatar a su mascota dragón, superando obstáculos, luchando contra enemigos y explorando diversos mundos fantásticos.

Proyecto 2: Aplicación ToDo

Una aplicación de gestión de tareas con integración de calendario. Organiza tus actividades diarias, establece recordatorios y marca tareas como completadas. Además, contará con opciones de integración para sincronizar con otras plataformas.

Proyecto 3: Juego de Blackjack

Un juego clásico de Blackjack donde los jugadores podrán apostar y jugar contra la banca. Este juego estará disponible para plataformas móviles y web, ofreciendo una experiencia sencilla pero divertida de cartas.
